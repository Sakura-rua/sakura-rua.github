节点搭建

SS | SSR

一键脚本搭建SS/搭建SSR服务并开启BBR加速

安装 
yum -y install git//安装Git//Centosapt-get update && apt-get -y install git//Debiangit clone https://github.com/suniceman/ss-fly//一键搭建SS脚本ss-fly/ss-fly.sh -i password port//运行搭建SS//password和port自定义
相关操作 
/etc/init.d/ss-fly start //启动
/etc/init.d/ss-fly stop //停止
/etc/init.d/ss-fly restart //重启
/etc/init.d/ss-fly status //状态
ss-fly/ss-fly.sh -sslink //查看ss链接
vim /etc/shadowsocks.json //修改配置文件
ss-fly/ss-fly.sh -uninstall //卸载SS
