一键安装脚本

apt-get install wget && wget https://raw.githubusercontent.com/xtaodada/PagerMaid-Modify/master/install.sh && chmod +x install.sh && bash install.sh

脚本已测试系统：centos8、centos7、ubuntu16.04、ubuntu18.04、ubuntu20.04、debian9、debian10
理论支持系统：centos8 及以下、ubuntu20.04 及以下、debian10 及以下

链接数据库 apt-get install redis-server -y && redis-cli 连上后 ctrl + d 返回正常界面
